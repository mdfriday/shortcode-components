强化IP结构

痛点 + 用户获得感 + IP信任感 + 解决方案

1. 痛点

将痛点放到视频开头的好处是，吸引来的都是精准粉丝，所以要思考你的目标群体的真正痛点！

例如：如何免F读书还能每个月赚 23000+？年轻人如何不靠打工才能够养活自己？

2. 获得感

你需要在 2 秒钟内告诉用户我将会为你提供巨大的价值，例如速成类知识：

3 步教大家快速学习新的赚钱技能，5 分钟教会你 7 种减脂做法，简单好吃！

3. 信任感

强化你将要陈述的内容的可信度，以及你本人的可信度/别人为什么信任你，同时也是为了吸引用户看下去的钩子，递进式塑造期待感。

例如：我曾经帮助 2000+ 学员解决某些问题，曾经获得官方的某些认证/奖励/技能。

4. 解决方案

价值是目前所有铺垫工作的最终结果，同时也是观众是否点赞关注的重要因素，如果故事夸大虚无只会让用户厌烦。

例如：盘点真正有用的信息，展示卖点 1 / 卖点 2 / 卖点 3。
